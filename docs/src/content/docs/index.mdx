## 特徴

<CardGrid>
  <Card title="無線左右分割キーボード">
    自由な手の位置でタイピングできます。

    裏側にはマグネットが付いており、持ち運びを考えた設計になっています。

  </Card>
  <Card title="25mm トラックボール内蔵">
    マウスを持ち替えることなく、カーソル操作が可能です。

    外出先でもトラックボールを使った快適な操作ができます。

  </Card>
  <Card title="タッチセンサー搭載">
    トラックボールのスクロールモード切り替えなどに使えるタッチセンサーを左右に2つずつ搭載しています。
  </Card>
  <Card title="電池駆動" >
    左右はそれぞれ単４電池１本ずつで動作します。

    リチウムポリマーバッテリーと比べて発火の心配が低く、安心して使用できます。

  </Card>
  <Card title="配列">
    Column Staggered 配列の美しさと使い慣れた Row Staggered 配列の特徴を兼ね備えています。

    Column Staggered 配列が合わない方でも違和感が少なく使えるように設計されています。

  </Card>
  <Card title="自由なカスタマイズ">
    ZMKファームウェアを使っておりソースコードも公開しているので、キーマップやマクロ、細かな振る舞いを自由にカスタマイズできます。
  </Card>
</CardGrid>
